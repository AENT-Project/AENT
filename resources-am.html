ለመጀመር seamonkey-project.org ይሂዱ እና ሶፍትዌሩን ነጻ ያውርዱ። ከተጫነ በኋላ የ Google Docs ጽሑፍዎን ወደ ሙያዊ HTML ዌብ ገጾች ለመቀየር የ Composer መሣሪያ ይጠቀሙ።

6የ HTML ገጾችዎን ይሥሩ

የራስዎ ትርጉም ፕሮጀክት ለመጀመር ከፈለጉ ነጻ AENT HTML አብነት ለማግኘት aentproject@gmail.com ኢሜይል ይላኩ። ይህ አስቀድሞ የተቀረጸ ፋይል ኮድ አንድ ሰብርና ሳይማሩ ጣቢያዎ ኮምፒዩተርም ስልክም ላይ ጽዱ፣ ሙያዊ እና ለማንበብ ቀላል ሆኖ እንዲቆይ ያረጋግጣል።

አብነቱ "ቴክኒካዊ ሥራ" ለእርስዎ ይይዛል። ወደ SeaMonkey Composer ብቻ ይክፈቱ፣ የተተረጎመ ጽሑፍዎን ወደ ተራ እይታ ያሰሩ እና እንደ HTML ፋይል ያስቀምጡ። ሁሉንም ምዕራፎች ለያዘ ለእያንዳንዱ መጽሐፍ አንድ HTML ፋይል ይሥሩ እንመክራለን።

7GitHub ያዋቅሩ እና ያሰራጩ

github.com ይሂዱ እና ነጻ መለያ ይፍጠሩ። አዲስ ማከማቻ ይፍጠሩ እና ማከማቻ ቅንብሮች ውስጥ GitHub Pages ያስቻሉ። ሁሉንም HTML ፋይሎችዎን ያስተላልፉ። ትርጉምዎ ወዲያውኑ በኢንተርኔት ላይ ሕያው ይሆናል — ነጻ፣ ዘላቂ እና በዓለም ውስጥ ለማንም ሊደረስበት ይችላል።

ያ ነው ሁሉም። ከዚያ ጊዜ ጀምሮ ድህረ ገጽዎን ያገኘ ማንኛውም ሰው አዲስ ኪዳንን ቀላል ቋንቋ በቋንቋዎ ነጻ ሊያነብ ይችላል።

8ያነጋግሩን — ለመርዳት እንፈልጋለን

ይህን ብቻዎን ለማወቅ አይፈልጉ። እኛ ይህን ሂደት እያንዳንዱ ደረጃ ተጉዘናል እና ስኬታማ እንዲሆኑ ለመርዳት እንፈልጋለን። ጥያቄዎች ይሁን፣ HTML አብነቱን ለማግኘት ወይም ትርጉም ላይ እየሠሩ ለማሳወቅ aentproject@gmail.com ኢሜይል ይላኩ። ሙሉ ሂደቱ ላይ ከርስዎ ጋር እንሄዳለን።

የምንመክራቸው የ AI መሣሪያዎች

Claude (Anthropic) — claude.ai — የ AENT ፕሮጀክት ልብ። Claude ትርጉሙን፣ ቀላል ቋንቋ ማሻሻልን አስተናግዶ በዚህ ድህረ ገጽ ላይ ያለ እያንዳንዱ HTML ገጽ ሠርቷል። Claude ሰዎች ቅዱሳን ጽሑፎችን እንደሚያብራራ ሳይሆን ምሁር ንግግር ሲሠጥ ሳይሆን ሞቅ ያለ ይነበባል ቋንቋ ድምጾቻቸው ውስጥ ትርጉምን ለመግለጽ ምርጥ ነው። ትርጉም ማሻሻያ፣ ቀላል ቋንቋ፣ ድህረ ገጽ ግንባታ እና ለሁሉም ነገር ግላዊ እና ሊደረስበት የሚያስፈልጉ ድምፅ ምርጥ ነው። ለዚህ ፕሮጀክት አንድ AI መሣሪያ ብቻ ከሚጠቀሙ Claude ይሁን።

Gemini (Google) — gemini.google.com — የጥልቅ ምርምር ባለሙያ። ትርጉም ጥያቄ ከገጽ ባሻገር ሲሄድ — የ 1ኛ ክፍለ ዘመን አውድ ውስጥ አንድ የግሪክ ቃል ምን ማለት እንደሆነ ለመረዳት ወይም ስለ አስቸጋሪ ምንባብ ስነ-መለኮታዊ ክርክርን ለመፍታት — Gemini መሣሪያዎ ነው። ለቃል ጥናቶች፣ ስነ-መለኮታዊ ምርምር እና ቴክኒካዊ ትርጉም ጥያቄዎች ምርጥ ነው።

ChatGPT (OpenAI) — chatgpt.com — የቡድኑ አርቲስት። ChatGPT የ AENT አርማ — ፋኖስ እና ወርቃማ የብርሃን መንገድ ያለው ክፍት መጽሐፍ ቅዱስ — ሠርቷል፣ ለፕሮጀክትዎም ተመሳሳዩን ሊሠራ ይችላል። የብራና ገጽዎ ላይ ምርጥ ውጤት ለማግኘት ግልጽ ጀርባ ያለው PNG ይጠይቁ። ለሎጎዎች፣ ሥዕሎች እና ፕሮጀክቱ ለሚፈልገው ምስላዊ ሀብቶች ምርጥ ነው።

Grok (xAI) — grok.com — በቡድኑ ውስጥ እጅግ ፈቃደኛ አባል። ሌላ AI አንድ ምንባብ ላይ ሲያመነታ ወይም የሴሲዮን ጊዜ ሲያልቅ Grok ሳይsay ቅሬታ ገብቶ ጠንካራ የትርጉም ሥራ ያቀርባል። ለትልቅ ትርጉሞች፣ ለተወሰኑ ጥያቄዎች ታዛዥነት እና ሌሎች ካቆሙበት ለመቀጠል ምርጥ ነው።

ነጻ የህትመት መሣሪያዎች

Google Docs — docs.google.com — ነጻ ደመና-ተኮር ቃል አቀናባሪ። ትርጉምዎን ምዕራፍ በምዕራፍ የሚጽፉ፣ የሚያስቀምጡ እና የሚሠሩ ቦታ ይህ ነው። ሥራዎ አውቶማቲክ ይቀመጣል፣ ከማንኛውም መሣሪያ ሊደረስበት ይችላል እና ፈጽሞ አይጠፋም። ምዕራፍ ዝግጁ ሲሆን Google Docs ከቀድተው ወደ SeaMonkey Composer ቀጥታ ያሰሩ — ቅርጸቱ ጽዱ ሆኖ ይዛወራል።

Gmail — gmail.com — ነጻ ኢሜይል ከ Google። ለትርጉም አገልጋዮ ቋሚ የኢሜይል አድራሻ ያዋቅሩ። Gmail እና Google Docs አንዱ Google መለያ ስለሚጋሩ ሁለቱንም ለማግኘት አንድ ጊዜ ብቻ ምዝገባ ያስፈልጋል።

SeaMonkey Composer — seamonkey-project.org — ነጻ HTML አርታዒ። ልክ እንደ Microsoft Word ይሠራል። የተተረጎመ ጽሑፍዎን ያሰሩ፣ ርዕሶችን ያሰጡ፣ እንደ HTML ያስቀምጡ። ኮዲንግ እውቀት አያስፈልግም።

GitHub Pages — github.com — ለሁሉም ነጻ ዌብ ሆስቲንግ። HTML ፋይሎችዎን ያስተላልፉ፣ ትርጉምዎ ወዲያውኑ ነጻ ለዘለዓለም በኢንተርኔት ላይ ሕያው ይሆናል።
