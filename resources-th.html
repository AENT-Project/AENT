AENT

พระวจนะของพระองค์เป็นโคมสำหรับเท้าของข้าพเจ้า และเป็นแสงสว่างแก่ทางของข้าพเจ้า — สดุดี 119:105
หน้านี้มีให้บริการในภาษาอื่น ดูเป็นภาษาอังกฤษ

ทรัพยากร

ทุกสิ่งที่คุณต้องการเพื่อนำพระคัมภีร์ในภาษาที่เข้าใจง่ายไปสู่ประชาชนของคุณ — ฟรีทั้งหมด

หากคุณเป็นคนใหม่ อยากรู้อยากเห็น หรือเพียงแค่ต้องการความชัดเจน — เริ่มต้นที่นี่ แหล่งข้อมูลเหล่านี้มีไว้เพื่อช่วยให้คุณเข้าใจ — ไม่ใช่เพื่อทำให้คุณรู้สึกท่วมท้น

วิสัยทัศน์

AENT — พันธสัญญาใหม่ภาษาอังกฤษที่เข้าถึงได้ — ถูกสร้างขึ้นเพื่อพิสูจน์สิ่งที่เรียบง่าย พันธสัญญาใหม่ทั้งเล่มสามารถแปลเป็นภาษาที่ชัดเจนและเรียบง่ายและเผยแพร่ไปทั่วโลกได้ฟรีโดยใช้เครื่องมือที่ทุกคนสามารถเข้าถึงได้ ไม่จำเป็นต้องมีปริญญาจากเซมินารี ไม่จำเป็นต้องมีงบประมาณในการจัดพิมพ์ ไม่จำเป็นต้องมีพื้นฐานทางเทคนิค

เราสร้างฉบับภาษาอังกฤษจากศูนย์โดยใช้เครื่องมือ AI ฟรี ซอฟต์แวร์ฟรี และการโฮสต์เว็บฟรีเท่านั้น โครงการทั้งหมด — การแปล เว็บไซต์ และการเผยแพร่ — ไม่มีค่าใช้จ่ายใดๆ นอกจากเวลาและความซื่อสัตย์

ตอนนี้เราต้องการแสดงให้คุณเห็นว่าเราทำสิ่งนี้ได้อย่างไร เพื่อที่คุณจะได้ทำสิ่งเดียวกันนี้สำหรับประชาชนของคุณในภาษาของคุณ พระวจนะของพระเจ้าเป็นของทุกคน ในโลกของพระเจ้าเราทุกคนเป็นพี่น้องและเพื่อนบ้าน ทุกคนบนโลกนี้สมควรอ่านมันในภาษาที่พวกเขาเข้าใจจริงๆ จงรักพระเจ้าด้วยสุดใจสุดจิตสุดความคิดและสุดกำลังของคุณ แพร่กระจายข่าวดีของพระองค์ไปทุกหนทุกแห่ง

หลักการแปลของ AENT

นี่คือแนวทางที่เราปฏิบัติตามในทุกบทของทุกหนังสือ ปฏิบัติตามหลักการเหล่านี้และการแปลของคุณจะมีคุณภาพที่อบอุ่น ชัดเจน และอ่านง่ายเช่นเดียวกับฉบับภาษาอังกฤษ

  • ภาษาเรียบง่ายเสมอ หากนักเรียนมัธยมต้นไม่สามารถเข้าใจประโยคนั้นได้ ให้เขียนใหม่ อย่าใช้ภาษาโบราณหรือภาษาทางศาสนาเมื่อมีคำเรียบง่ายอยู่
  • ใช้ชื่อที่คุ้นเคย ใช้ชื่อที่คนรู้จักอยู่แล้ว — พระเยซู ไม่ใช่ Yeshua พระเจ้า ไม่ใช่ Elohim เป้าหมายคือความเข้าใจ ไม่ใช่ความรู้ทางวิชาการ
  • หนึ่งความคิดต่อหนึ่งประโยค ประโยคยาวและซับซ้อนทำให้ผู้อ่านหลงทาง แบ่งออก ให้ชัดเจน
  • พระเจ้าด้วยตัวพิมพ์ใหญ่ เมื่อข้อความอ้างถึงพระนามส่วนตัวอันศักดิ์สิทธิ์ของพระเจ้า ให้เขียน องค์พระผู้เป็นเจ้า ด้วยตัวพิมพ์ใหญ่ เมื่อเป็นคำยกย่อง ให้เขียนพระเจ้าด้วยตัวอักษรธรรมดา
  • กรอบการเป็นเพื่อนคู่คิด การแปลนี้เป็นเพื่อนคู่คิดของพระคัมภีร์ที่ผู้อ่านชื่นชอบ — ไม่ใช่สิ่งทดแทน บอกสิ่งนี้อย่างชัดเจนที่ต้นของแต่ละหนังสือ
  • เสียงที่อบอุ่นและอ่านง่าย เขียนในแบบที่เพื่อนที่ไว้วางใจได้จะอธิบายพระคัมภีร์ — ไม่ใช่แบบที่นักวิชาการเขียนคำอธิบาย
  • ซื่อสัตย์ต่อความหมาย ภาษาเรียบง่ายไม่ได้หมายความว่าเป็นการถอดความอย่างเสรี การเลือกแปลแต่ละครั้งควรให้เกียรติความหมายของข้อความภาษากรีกดั้งเดิม

ทีละขั้นตอน — วิธีสร้างฉบับของคุณเอง

1สร้างบัญชี Google ฟรีของคุณ

ไปที่ google.com และสร้างบัญชีฟรี บัญชีเดียวนี้ให้ Gmail สำหรับที่อยู่อีเมลโครงการของคุณและ Google Docs สำหรับเขียนและจัดเก็บการแปลของคุณ ตั้งชื่อที่อยู่อีเมลให้ชัดเจน — เช่น thaiaent@gmail.com หรือ aentthai@gmail.com

2รับข้อความต้นฉบับของคุณ

คุณต้องการพันธสัญญาใหม่ภาษากรีกที่เชื่อถือได้เป็นพื้นฐาน เราแนะนำ SBL Greek New Testament — ค้นหาออนไลน์ว่า "SBL Greek New Testament free PDF" เพื่อค้นหา คุณไม่จำเป็นต้องอ่านภาษากรีกด้วยตัวเอง เครื่องมือ AI จะจัดการการแปลจากภาษากรีก คุณเพียงแค่ต้องการข้อความต้นฉบับที่พร้อมใช้งานเป็นข้อมูลอ้างอิง

3เปิด Grok หรือ Gemini และเริ่มแปล

ไปที่ grok.com หรือ gemini.google.com และสร้างบัญชีฟรี เครื่องมือทั้งสองยอดเยี่ยมสำหรับงานแปลและมีระดับฟรีที่ใจกว้างซึ่งจะนำคุณผ่านพันธสัญญาใหม่ทั้งเล่มโดยไม่หมดเวลาเซสชัน

สำหรับแต่ละบทให้คำสั่ง Grok หรือ Gemini ดังนี้ — "โปรดแปลมัทธิว บทที่ 1 จากพันธสัญญาใหม่ภาษากรีกเป็นภาษาไทยที่เรียบง่ายโดยปฏิบัติตามหลักการเหล่านี้ — ใช้ชื่อที่คุ้นเคยเช่นพระเยซูและพระเจ้า เขียนในระดับนักเรียนมัธยมต้น หนึ่งความคิดต่อหนึ่งประโยค เสียงที่อบอุ่นและเป็นมิตร องค์พระผู้เป็นเจ้าด้วยตัวพิมพ์ใหญ่เมื่ออ้างถึงพระนามส่วนตัวของพระเจ้า นี่คือการแปลเพื่อเป็นเพื่อนคู่คิด ไม่ใช่สิ่งทดแทนพระคัมภีร์ที่มีอยู่"

ตรวจสอบแต่ละบทอย่างละเอียด ปรับแต่งสิ่งที่ไม่ฟังดูเป็นธรรมชาติในภาษาของคุณ วางบทที่เสร็จแล้วใน Google Docs บันทึก ไปยังบทถัดไป

เมื่อการแปลของคุณเสร็จสมบูรณ์และพร้อมที่จะสร้างเว็บไซต์ — นั่นคือเวลาที่คุณนำ Claude มาที่ claude.ai Claude ยอดเยี่ยมในการสร้างหน้า HTML ปรับแต่งภาษาเรียบง่าย และแก้ปัญหาทางเทคนิค ใช้แต่ละเครื่องมือสำหรับสิ่งที่ทำได้ดีที่สุด

4ทำงานทีละเล่ม

แปลทีละเล่ม เริ่มด้วยพระวรสารมาระโก — เป็นพระวรสารที่สั้นที่สุดและเรื่องราวที่ตรงที่สุด หลังจากเสร็จสิ้นมาระโกคุณจะมีความรู้สึกที่แข็งแกร่งต่อเสียงของการแปลและส่วนที่เหลือจะดำเนินไปเร็วขึ้น

จัดเก็บแต่ละเล่มที่เสร็จแล้วเป็นเอกสารแยกต่างหากใน Google Docs งานของคุณจะถูกบันทึกโดยอัตโนมัติในระบบคลาวด์และสามารถเข้าถึงได้จากทุกอุปกรณ์ จะไม่สูญหายเลย

5ดาวน์โหลด SeaMonkey Composer

สำหรับการแก้ไขเว็บไซต์ทั้งหมดเราแนะนำ SeaMonkey เป็นเครื่องมือสไตล์เก่าที่ทรงพลังซึ่งทำงานเหมือน Microsoft Word ทุกประการ หากคุณสามารถคัดลอกข้อความจากเอกสารและวางในหน้าต่างใหม่ได้คุณก็มีทักษะในการสร้างเว็บไซต์ด้วย SeaMonkey แล้ว ไม่ต้องการความรู้การเขียนโค้ดเลย

หากต้องการเริ่มต้นไปที่ seamonkey-project.org และดาวน์โหลดซอฟต์แวร์ฟรี หลังจากติดตั้งแล้วให้ใช้เครื่องมือ Composer เพื่อแปลงข้อความ Google Docs ของคุณเป็นหน้าเว็บ HTML ระดับมืออาชีพ

6สร้างหน้า HTML ของคุณ

หากคุณต้องการเริ่มต้นโครงการแปลของคุณเองให้ติดต่อเราที่ aentproject@gmail.com เพื่อรับ แม่แบบ HTML AENT ฟรี ไฟล์ที่จัดรูปแบบล่วงหน้านี้ช่วยให้แน่ใจว่าไซต์ของคุณยังคงสะอาด เป็นมืออาชีพ และอ่านง่ายทั้งบนคอมพิวเตอร์และโทรศัพท์ — โดยไม่ต้องเรียนรู้โค้ดแม้แต่บรรทัดเดียว

แม่แบบจะจัดการ "งานทางเทคนิค" ให้คุณ เพียงแค่เปิดใน SeaMonkey Composer วางข้อความที่แปลแล้วใน มุมมองปกติ และบันทึกเป็นไฟล์ HTML เราแนะนำให้สร้างไฟล์ HTML หนึ่งไฟล์ต่อหนังสือหนึ่งเล่มที่ประกอบด้วยทุกบท

7ตั้งค่า GitHub และเผยแพร่

ไปที่ github.com และสร้างบัญชีฟรี สร้าง repository ใหม่และเปิดใช้งาน GitHub Pages ในการตั้งค่า repository อัปโหลดไฟล์ HTML ทั้งหมดของคุณ การแปลของคุณจะออนไลน์บนอินเทอร์เน็ตทันที — ฟรี ถาวร และทุกคนในโลกสามารถเข้าถึงได้

นั่นแหล่ะ ตั้งแต่ช่วงเวลานั้นเป็นต้นไปทุกคนที่พบเว็บไซต์ของคุณสามารถอ่านพันธสัญญาใหม่ในภาษาเรียบง่ายในภาษาของคุณได้ฟรี

8ติดต่อเรา — เราต้องการช่วยเหลือ

คุณไม่จำเป็นต้องค้นหาทุกอย่างด้วยตัวเอง เราผ่านทุกขั้นตอนของกระบวนการนี้และต้องการช่วยให้คุณประสบความสำเร็จ ส่งอีเมลหาเราที่ aentproject@gmail.com พร้อมคำถามเพื่อรับแม่แบบ HTML หรือเพียงแค่แจ้งให้เราทราบว่าคุณกำลังทำงานในการแปล เราจะเดินทางร่วมกับคุณตลอดกระบวนการ

เครื่องมือ AI ที่เราแนะนำ

Claude (Anthropic) — claude.ai — หัวใจของโครงการ AENT Claude จัดการการแปล การปรับแต่งภาษาเรียบง่าย และสร้างทุกหน้า HTML บนเว็บไซต์นี้ Claude ยอดเยี่ยมในความแตกต่างละเอียดอ่อน โทนเสียง และการแสดงความหมายในภาษาที่อบอุ่นและอ่านง่ายที่รู้สึกเหมือนเพื่อนที่ไว้วางใจได้อธิบายพระคัมภีร์แทนที่จะเป็นนักวิชาการบรรยาย ดีที่สุดสำหรับการปรับแต่งการแปล ภาษาเรียบง่าย การสร้างเว็บไซต์ และทุกสิ่งที่ต้องการเสียงส่วนตัวและเข้าถึงได้ หากคุณจะใช้เครื่องมือ AI เดียวสำหรับโครงการนี้ให้เป็น Claude

Gemini (Google) — gemini.google.com — ผู้เชี่ยวชาญด้านการวิจัยเชิงลึก เมื่อคำถามการแปลเกินกว่าพื้นผิว — เมื่อคุณต้องการเข้าใจว่าคำภาษากรีกหมายความว่าอะไรในบริบทของศตวรรษแรกหรือแก้ไขการโต้เถียงทางเทววิทยาเกี่ยวกับข้อความที่ยาก — Gemini คือเครื่องมือของคุณ ดีที่สุดสำหรับการศึกษาคำ การวิจัยทางเทววิทยา และคำถามการแปลทางเทคนิค

ChatGPT (OpenAI) — chatgpt.com — ศิลปินของทีม ChatGPT สร้างโลโก้ AENT — พระคัมภีร์ที่เปิดพร้อมตะเกียงและเส้นทางแห่งแสงสีทอง — และสามารถทำสิ่งเดียวกันสำหรับโครงการของคุณ ขอ PNG ที่มีพื้นหลังโปร่งใสเพื่อผลลัพธ์ที่ดีที่สุดบนหน้าหนังสัตว์ของคุณ ดีที่สุดสำหรับโลโก้ ภาพประกอบ และสินทรัพย์ภาพที่โครงการของคุณต้องการ

Grok (xAI) — grok.com — สมาชิกที่เต็มใจที่สุดในทีม เมื่อ AI อื่นลังเลในข้อความเฉพาะหรือหมดเวลาเซสชัน Grok จะเข้ามาโดยไม่บ่นและส่งมอบงานแปลที่มั่นคง ดีที่สุดสำหรับการแปลปริมาณมาก การปฏิบัติตามคำขอเฉพาะ และการดำเนินการต่อจากที่คนอื่นหยุด

เครื่องมือเผยแพร่ฟรี

Google Docs — docs.google.com — โปรแกรมประมวลผลคำฟรีบนคลาวด์ นี่คือที่ที่คุณเขียน จัดเก็บ และขัดเกลาการแปลของคุณทีละบท งานของคุณจะถูกบันทึกโดยอัตโนมัติ เข้าถึงได้จากทุกอุปกรณ์ และจะไม่สูญหายเลย เมื่อบทพร้อมแล้วเพียงคัดลอกจาก Google Docs และวางโดยตรงใน SeaMonkey Composer — การจัดรูปแบบจะถ่ายโอนอย่างสะอาด

Gmail — gmail.com — อีเมลฟรีจาก Google ตั้งค่าที่อยู่อีเมลเฉพาะสำหรับพันธกิจการแปลของคุณ เนื่องจาก Gmail และ Google Docs ใช้บัญชี Google เดียวกัน คุณจึงต้องลงทะเบียนเพียงครั้งเดียวเพื่อรับทั้งสอง

SeaMonkey Composer — seamonkey-project.org — โปรแกรมแก้ไข HTML ฟรี ทำงานเหมือน Microsoft Word วางข้อความที่แปลแล้ว ใส่หัวเรื่อง บันทึกเป็น HTML ไม่ต้องการความรู้การเขียนโค้ด

GitHub Pages — github.com — การโฮสต์เว็บฟรีสำหรับทุกคน อัปโหลดไฟล์ HTML ของคุณและการแปลของคุณจะออนไลน์ทันทีฟรีตลอดไป
